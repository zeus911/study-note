Git   暂存区
===
###  工作区 暂存区 版本库

```
工作区-------------------------------暂存区----------------------------------版本库
	-> add 			         -> commit	
	<- checkout                                 <- reset HEAD
	--< rm --cached
`checkout HEAD` 从版本库还原文件覆盖暂存区和工作取.	
```
### Diff

```
工作区---------`git diff`------------暂存区
工作区----------------------------------`git diff HEAD/master`------------------版本库
				暂存区--------`git diff cached`------版本库
```
